# 説明動画 制作計画

## 目的と用途

| 用途 | 形式 | 対象 | 尺の目安 |
|---|---|---|---|
| ユーザー向け・LP埋め込み | **ステップ別 × 5本** | T4プレイヤー | 各30秒〜1分 |
| ユーザー向け・SNSシェア | ダイジェスト 1本 | 新規ユーザー | 3分 |
| ポートフォリオ | 技術解説込み 1本 | 採用担当・クライアント | 7〜8分 |

**制作方針：録画は1回。ステップ別クリップを結合してダイジェスト・ポートフォリオ版を作る。**

```
ステップ別クリップ（STEP 01〜05）× 5本  ← LP の各ステップに埋め込み
          ↓ 結合
ダイジェスト版 1本（3分）              ← SNS・共有用
          ↓ 技術シーンを追加
ポートフォリオ版 1本（7〜8分）         ← 採用・クライアント向け
```

---

## 5STEP（LP・動画共通構成）

| STEP | タイトル | LP現行からの変更 |
|---|---|---|
| 01 | 拡張機能をインストール | 変更なし |
| 02 | Googleログイン → T4でプレイ（自動蓄積） | 旧02+旧03を統合 |
| 03 | セッション画面で解析 → PDF出力 | PDF出力を追記 |
| 04 | AI解説で深掘り | 「任意」表記を削除 |
| 05 | 3D可視化で傾向を掴む | 新規追加 |

## 録画前チェックリスト

### プライバシー確認（必須）
- [ ] Groq APIキーをダミーに差し替え（設定画面録画前）
- [ ] Googleアカウントのメールアドレスが映り込まない設定を確認
- [ ] URLのUID部分が長時間映らないよう注意

### デモデータ確認（必須）
- [ ] 解析済みハンドが十分にある（グラフ・3Dが映える状態）
- [ ] AI解析結果が少なくとも1件ある

### OBS設定
- [x] 基本・出力解像度：1920×1080
- [x] フレームレート：30fps
- [x] ソース：ウィンドウキャプチャ（Chrome）
- [x] バウンディングボックス：境界の内側に合わせる（黒帯あり・歪みなし）
- [ ] マウスカーソルを目立たせる設定をON

### OBSホットキー（設定済み）
| キー | 動作 |
|---|---|
| o | 録画開始 |
| P | 録画終了 |
| K | 録画一時停止 |
| L | 録画再開 |

## 録画時のルール

**操作の前後に必ず1〜2秒停止する。**
字幕・ナレーションを後付けするため、操作の直前と直後に間がないと編集不能になる。
ボタンを押す前に止まる → 押す → 結果が出たら止まる、の3拍子を意識する。
